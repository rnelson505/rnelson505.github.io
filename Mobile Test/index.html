Projects

June, 2019

'Tour of California' Cycling Map

This map was created using ArcGIS Pro, Adobe Illustrator and Adobe Photoshop. The routes were generated using Google Earth and brought into Adobe Illustrator for styling. DEM's and landcover data were brought into Adobe Photoshop to create the hillshade effect using blending mode and other tools. All vector data and images were modified in Illustrator.

May, 2019

Bivariate Map + Infographic

This is my long-form infographic which was created using ArcGIS and Adobe Illustrator. It contains a bivariate map along with some statistical data showing the relationship between gun ownership and suicide. This was my first project using Adobe Illustrator.

May, 2018

Site Suitability Analysis

This was my final research project for an undergraduate class which introduced me to GIS for environmental analysis using mostly raster-based geoprocessing tools in ArcGIS. This map is a site-suitability analysis for transit-oriented infill development in the Oakland, CA area.

February, 2018

Dasymetric Population Density

One of the earlier maps I have created using ArcGIS, this choropleth map more accurately estimates populated areas in Alameda County, CA by using land-use data to eliminate land-use categories where populations do not normally exist.
